About Manulife - Hybrid Automation Test Portal!!!

 Manulife - Hybrid Automation Test Portal is a Automation Framework tool designed to perform Sanity and Functional Testing for iOS Mobile and Web application. User can run Pre-designed Automated Test scripts with single click from Mobile or Desktop Browser and Script will be executed on Pre-configured Test Lab and Results will be sent to User in Email. This Portal is useful for QA or BA Team to Test E2E Functional Workflow of Application.

Benefits of Automation Portal

 User Friendly - Automation Portal Portal offers the user friendly single clicks to run Automated Test Scripts without any manual intervention.

  Auto-Email Results – Portal helps the QA or BA Team in getting reports generated and notified in an Auto-email with Test Execution Results.

  A la carte - QA or BA Team can choose the required E2E Functional Test to be executed.

  Test Data – QA or BA Team does not need to enter any test data for Sanity Scripts.

  Multiple Environment Testing - User can execute Test cases in different environments. Example: SIT and UAT.

  Cross Platform Testing - User can execute Test cases in different platform. Example: iOS Mobile App Or Web Application.
